This demonstration uses CSS animation effects to produce an animated guide experience over a web page. About the Demo: Div Transitions

The concept is that any web page can become a stage for animated effects. Div Transitions creates the concept of an observer. The overserver, in this case, is a DIV element and its' imediate children receives events of transformations, thus causing the effect of observation. It explores the gap between motion picture and web pages, it aims to be a system that can allow users to toggle between the normal desktop view of a web page and a new state, an in-motion state.
