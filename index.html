TagVisor.js
by Taboca and TelaSocial Labs. TagVisor first release was featured as part of a devDerby demonstration — conductor — July 2011.
We are here to help you transform your web pages into animation mode preserving the existing markup!
You can add a few animation rules to your web page and you have this nice sliding effect.
And 3D also works, if you have a newer browser but it works anyways if CSS 3D transformation is not in place.
TagVisor is a JavaScript library that can turn your web page into animation mode without extra markup. You can preserve the page as it is and you can tell stories on top of them. To fork the project in GitHub or to find samples please use the Github project page.
